About the System

Smart Street Light Simulation System

System Information

The Smart Street Light Simulation System is a technology-driven prototype designed to model, simulate, and visualize the operation of an intelligent street lighting network using geographic data. The project integrates modern web technologies with geospatial mapping to replicate real-world street light behaviors and conditions in a controlled virtual environment.

The simulation system aims to replicate a functional smart lighting environment where each street light is represented on an interactive digital map according to its geographic coordinates. Users can view, monitor, and manage light statuses (e.g., ON, OFF, or FAULT) through a web-based dashboard interface.

Key Features:

  • Real-time status monitoring
  • Interactive map visualization with Mapbox
  • Fault detection and alerts
  • Energy-efficient management
  • Color-coded status indicators
